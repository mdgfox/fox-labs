# Green Sock (GSAP) Docs

## What is GSAP?
    GSAP is a robust JavaScript toolset that turns developers into animation superheroes.
    Build high-performance animations that work in every major browser.
    Animate CSS, SVG, canvas, React, Vue, WebGL, colors, strings, motion paths, generic objects...anything
    JavaScript can touch! GSAP's ScrollTrigger plugin lets you create jaw-dropping scroll-based animations
    with minimal code. gsap.matchMedia() makes building responsive, accessibility-friendly animations a breeze.
